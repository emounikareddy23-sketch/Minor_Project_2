# 💳 Minor Project 2 – SpendDNA: Bank Transaction Analysis using Python

## 📌 Project Overview

SpendDNA is a Python-based data analytics project that analyzes bank transaction data to generate meaningful financial insights. The project focuses on data cleaning, preprocessing, exploratory data analysis (EDA), spending behavior analysis, anomaly detection, and report generation using Python.

The objective is to transform raw banking transaction data into actionable insights that help understand spending habits and financial trends.

---

## 🎯 Objectives

* Clean and preprocess bank transaction data.
* Handle missing values and duplicate records.
* Standardize date and time formats.
* Categorize transactions into spending categories.
* Analyze income, expenses, and savings.
* Identify monthly spending trends.
* Perform time-of-day spending analysis.
* Detect unusual (anomalous) transactions.
* Generate an executive summary report.

---

## 📂 Project Workflow

### Step 1: Data Import

* Load the transaction dataset.

### Step 2: Data Cleaning

* Remove duplicate records.
* Handle missing values.
* Convert data into appropriate formats.

### Step 3: Data Preprocessing

* Standardize transaction details.
* Convert date and time columns.
* Create useful derived features.

### Step 4: Exploratory Data Analysis (EDA)

* Debit vs Credit analysis
* Total spending
* Total income
* Net savings
* Savings rate

### Step 5: Spending Analysis

* Merchant-wise spending
* Category-wise spending
* Monthly spending trends
* Time-of-day transaction analysis

### Step 6: Anomaly Detection

* Identify unusually high-value transactions.
* Detect abnormal spending behavior.

### Step 7: Executive Report

Generate a summary including:

* Total Credits
* Total Debits
* Net Savings
* Savings Rate
* Top Spending Categories
* Monthly Trends
* Financial Insights
